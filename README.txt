# Интерактивная система продаж цемента

## Что делает
Читает заказы из файла, считает скидку (50% → 45% → ...), записывает результат.

## Структура
- `model/` — данные (Order, OrderResult)
- `reader/` — чтение файлов (txt с |, без расширения с #)
- `processor/` — расчёт скидки
- `writer/` — запись результата
- `manager/` — связка всего
- `Main.java` — точка входа

## Как запустить
java Main input.txt output.txt 10.0 50.0 5.0
